# Sela - Team Task Management
> A real-time, large-scale workspace for seamless team collaboration and dynamic project management.
Sela is a high-performance project management platform built to help modern teams collaborate in real-time. By bridging the gap between instant messaging and task tracking, Sela ensures team members can co-work on shared workspaces, track historical revisions, and organize workflows with fluid interactive controls.
## 🚀 Core Features (MVP)
- **Real-Time Collaborative Workspaces:** See live user profile cursors, presence indicators, and synchronous workspace updates.
- **Drag & Drop Operations:** Effortlessly organize, reorder, and move tasks across board columns.
- **History Versions Control:** Keep track of changes, visual logs, and rollback task states.
- **Fast & Seamless Workflow:** Built for ultra-low latency updates and quick workspace interactions.
## 🛠️ Tech Stack
- **Frontend:** React (Vite), Zustand (State Management), TanStack Query, Zod (Validation)
- **Backend:** Express.js, Socket.IO (Real-time), JWT Auth, Drizzle ORM, Redis (Caching/Pub-Sub)
- **Database:** PostgreSQL
